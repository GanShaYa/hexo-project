正则表达式

2022-01-20 0 评论 字数统计: 1.1k(字) 阅读时长: 4(分)

基本语法

^      匹配输入字符串开始位置
$      匹配输入字符串结束位置
()     子表达式开始和结束位置
.      匹配除换行符\n之外的任何单字符
A*     匹配任意个数字符                  空，A，AA，AAA
A+     匹配至少一次或多次字符             A，AA，AAA
A?     匹配0个或1个字符                  空，A
A{3}   指定个数字符                      AAA
A{2,3} 指定范围个数字符                  AA，AAA
A{2,}  至少n个字符                       AA，AAA，AAAA，...
A{0,3} 最多n个字符                       空，A，AA，AAA
[ABC]  范围内字符                        A，B，C
[A-F]  范围内任意字符                    A，B，C...F
[0-9]  范围内任意数字                    0，2，3...9
[^A-F] 反向范围内任意字符                非A-F
AB|CD  AB或CD                           AB，CD

匹配符

\\d    数字字符匹配
\\D    非数字字符匹配
\\w    匹配任何字类字符
\\W    匹配任何非字类字符
\\s    匹配任何空白字符
\\S    匹配任何非空白字符
\\f    换页符匹配
\\n    换行符匹配
\\r    匹配一个回车符
\\t    制表符匹配

是否匹配

String regex = "20\\d+";
"2022".matches(regex)  true
"1022".matches(regex)  false

String regex = "AB|CD";
"AB".matches(regex)  true
"AC".matches(regex)  false

分组匹配

正则表达式用(…)分组可以通过Matcher对象快速提取子串
Pattern p = Pattern.compile("(\\d{3,4})-(\\d{7,8})");
Matcher m = p.matcher("010-12345678");
if (m.matches()) {
  System.out.println(m.group(1)); // 010
  System.out.println(m.group(2)); // 12345678
} else {
     // 未匹配上
}

使用表达式搜索字符

String s = "the quick brown fox jumps over the lazy dog.";
Pattern p = Pattern.compile("\\wo\\w");
Matcher m = p.matcher(s);
while (m.find()) {
    String sub = s.substring(m.start(), m.end());
    System.out.println(sub);  // row
                                 fox
                                 dog
}

替换字符

String s = "The quick brown 123";
String r = s.replaceAll("\\d+", "fox");
System.out.println(r); // "The quick brown fox"

手机号脱敏
String replaceAll = "12345678912".replaceAll("(\\d{3})\\d{4}(\\d{4})", "$1****$2");
System.out.println(replaceAll); // 123****8912

非贪婪匹配

正则表达式匹配默认使用贪婪匹配，可以使用?表示对某一规则进行非贪婪匹配。
"非贪心的"模式匹配搜索到的、尽可能短的字符串，
"贪心的"模式匹配搜索到的、尽可能长的字符串。
在字符串"oooo"中，"o+?"只匹配单个"o"，而"o+"匹配所有"o"。

非捕获元

是一种零宽度的匹配，它匹配到的内容不会保存到匹配结果中去，最终匹配结果只是一个位置，代表字符串中的一个位置
作用：是给指定位置添加一个限定条件，用来规定此位置之前或者之后的字符必须满足限定条件才能使正则中的字表达式匹配成功
(?=表达式) 零宽正向先行断言：
紧接该位置之后的字符序列能够匹配
例：a regular expression   表达式  re(?=gular) 匹配结果：re
String s = "a regular expression";
Pattern p = Pattern.compile("re(?=gular)");
Matcher m = p.matcher(s);
if (m.find()) {
    String sub = s.substring(m.start(), m.end());
    System.out.println(sub);
}

(?!表达式) 零宽负向先行断言：
紧接该位置之后的字符序列不能匹配 
例：regex represents regular expression 表达式 re(?!g) 匹配结果 re  re  re
String s = "regex represents regular expression";
Pattern p = Pattern.compile("re(?!g)");
Matcher m = p.matcher(s);
while (m.find()) {
    String sub = s.substring(m.start(), m.end());
    System.out.println(sub);
}

(?<=表达式) 零宽正向后行断言：
紧接该位置之前的字符序列能够匹配
例：regex represents regular expression 表达式 (?<=\w)re 匹配结果 re  re
String s = "regex represents regular expression";
Pattern p = Pattern.compile("(?<=\\w)re");
Matcher m = p.matcher(s);
while (m.find()) {
    String sub = s.substring(m.start(), m.end());
    System.out.println(sub);
}

(?<!表达式) 零宽负向后行断言
紧接该位置之前的字符序列不能匹配 
例：regex represents regular expression 表达式 (?<\w)re 匹配结果 re  re  re
String s = "regex represents regular expression";
Pattern p = Pattern.compile("(?<!\\w)re");
Matcher m = p.matcher(s);
while (m.find()) {
    String sub = s.substring(m.start(), m.end());
    System.out.println(sub);
}
